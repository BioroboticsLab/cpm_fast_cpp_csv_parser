# cpm_fast_cpp_csv_parser

Installs [fast cpp_wsv parser](https://github.com/ben-strasser/fast-cpp-csv-parser)
with [cpm](https://github.com/iauns/cpm):

```
CPM_AddModule("cpm_fast_cpp_csv_parser"
    GIT_REPOSITORY "https://github.com/BioroboticsLab/cpm_fast_cpp_csv_parser"
    USE_EXISTING_VER TRUE)
```
